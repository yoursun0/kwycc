== Description ==
This theme designed by schiy and zwwooooo. It is a simple WordPress theme without any plugin needed (support plugin WP-PageNavi, WP-PostViews, wp-utf8-excerpt, WP-RecentComments), little images, custom-menu, widgets, threaded-comments. For WordPress version 3.0+.

== Installation ==

= Via WordPress Admin =
1. From your sites admin, go to Themes > Install Themes
1. In the search box, type 'River of Silver' and press enter
1. Locate the entry for 'River of Silver' (there should be only one) and click the 'Install' link
1. When installation is finished, click the 'Activate' link

= Manual Install =
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/river-of-silver/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the River of Silver listing on this page and click 'Activate'
